Luv2Store - Ecommerce Full-Stack

  • Objetivo:

    Poner en práctica nuevas herramientas tanto de front como backend

  • Descripción:

    Este proyecto consta de una tienda online con carrito de compras. En la página principal se encuentra: - Logotipo - Buscador de texto - Categorías de los productos - Cada producto con su imagen, nombre, precio y botón de añadir al carrito. - Soporte de paginación

  • Herramientas de Desarrollo:

    Front-End: Angular - TypeScript - HTML - CSS - BootStrap Back-End: Java - Spring Boot - Spring Data REST - API REST - Maven - Lombok - Hibernate - JPA - MySQL y PostreSQL
